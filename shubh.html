Create a table and explain how to acheive colspan

column1 column2 column3
data1 data2 data3
data4 data5 data6
data7 data8 data9

The colspan attribute in html specifies the number columns cell should span. it allow the single table cell to span the width of more than one cell or column.

It can used with <td> and <th> element while creating html table

It contain value i.e Number which specify the number of columns that a cell shoud span.

syntax: 
        <td colspan = "value">table content..</td>
